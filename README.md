# ▶ Tune In!
Listen to you favourite music with Tune In!
> A Spotify UI clone built with Vue.Js and Tailwind

## ✨ Features:
* Play, Pause and control the music you play ⏯
* Create Playlists 💞
* Search Page 🔍
